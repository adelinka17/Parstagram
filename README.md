# Parstagram -

This is a simplified Instagram clone with a custom Parse backend that allows a user to post photos, view a global photos feed, and add comments!

# Current Functionality
-  User stays logged in across restarts. 
-  User can log out.
- User can view comments on a post. 
-  User can add a new comment.



# Additional Actions that User can perform

- Sees app icon in home screen and styled launch screen. 
- Can sign up to create a new account.
- Can log in. 
- Can take a photo, add a caption, and post it to the server.
- Can view the last 20 posts
